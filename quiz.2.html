SQL Quiz 2

  1. ເລືອກຄຳຕອບທີ່ຖືກຕ້ອງ, ຈະໄດ້ຜົນລັບໃດຈາກ SQL Query ລຸ່ມນີ້:
    SELECT AVG(price) FROM products WHERE category = 'Electronics';
    1. ລາຄາສູງສຸດຂອງສິນຄ້າໃນປະເພດ Electronics
    2. ລາຄາຕໍາສຸດຂອງສິນຄ້າໃນປະເພດ Electronics
    3. ລາຄາສະເລ່ຍຂອງສິນຄ້າໃນປະເພດ Electronics
    4. ລາຄາລວມຂອງສິນຄ້າໃນປະເພດ Electronics
  2. ຂໍ້ໃດແມ່ນຜົນລັບຂອງ SQL Query ລຸ່ມນີ້:
    SELECT MIN(salary) FROM employees WHERE department = 'Sales';
    1. ເງິນເດືອນສູງສຸດຂອງພະແນກ Sales
    2. ເງິນເດືອນຕໍາສຸດຂອງພະແນກ Sales
    3. ເງິນເດືອນສະເລ່ຍຂອງພະແນກ Sales
    4. ເງິນເດືອນລວມຂອງພະແນກ Sales
  3. ຂໍ້ໃດແມ່ນຜົນລັບທີ່ຖືກຕ້ອງຂອງ SQL Query ລຸ່ມນີ້:
    SELECT e1.name, e1.salary FROM employees e1 WHERE e1.salary > (SELECT AVG(e2.salary) FROM employees e2 WHERE e2.department = e1.department);
    1. ລາຍຊື່ແລະເງິນເດືອນຂອງພະນັກງານທັງຫມົດ
    2. ລາຍຊື່ແລະເງິນເດືອນຂອງພະນັກງານທີ່ໄດ້ຮັບເງິນເດືອນສູງກວ່າເງິນເດືອນສະເລ່ຍໃນພະແນກຂອງຕົນ
    3. ລາຍຊື່ແລະເງິນເດືອນຂອງພະນັກງານທັງຫມົດທີ່ໄດ້ຮັບເງິນເດືອນເກີນ $50000
    4. ລາຍຊື່ແລະເງິນເດືອນຂອງພະນັກງານທັງຫມົດທີ່ຢູ່ໃນພະແນກການຕະຫຼາດ
  4. ຈົ່ງຈົດຈໍາເລືອກຄໍາຕອບທີ່ຖືກຕ້ອງຂອງ SQL Query ລຸ່ມນີ້:
    SELECT name FROM employees WHERE department = 'IT' AND salary > ALL (SELECT salary FROM employees WHERE department = 'HR');
    1. ລາຍຊື່ພະນັກງານໃນພະແນກ IT ທີ່ໄດ້ຮັບເງິນເດືອນສູງກວ່າພະນັກງານໃນພະແນກ HR
    2. ລາຍຊື່ພະນັກງານໃນພະແນກ IT ທີ່ໄດ້ຮັບເງິນເດືອນສູງສຸດ
    3. ລາຍຊື່ພະນັກງານໃນພະແນກ HR ທີ່ໄດ້ຮັບເງິນເດືອນສູງສຸດ
    4. ລາຍຊື່ພະນັກງານໃນພະແນກ IT ທີ່ໄດ້ຮັບເງິນເດືອນສູງກວ່າທຸກໆພະແນກ
  5. ຈົ່ງເລືອກຄຳຕອບທີ່ຖືກຕ້ອງຂອງ SQL Query ລຸ່ມນີ້:
    SELECT DISTINCT department FROM employees;
    1. ຊື່ທັງຫມົດຂອງພະແນກສຽງຂອງພະນັກງານ
    2. ຈຳນວນສິນຄໍາຂອງພະນັກງານໃນສະຖານທີ່ໃດສູງສຸດ
    3. ຊື່ທັງຫມົດຂອງພະແນກສຽງຂອງພະນັກງານ
    4. ຈຳນວນຂອງພະແນກທີ່ມີພະນັກງານໃນ
