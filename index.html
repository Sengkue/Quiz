1. ຈົ່ງເລືອກຄໍາຕອບທີ່ຖືກຕ້ອງ SQL Query ລຸ່ມນີ້ແມ່ນໃຊ້ໄວ້ເຮັດຫຍັງ:

SELECT name FROM employees WHERE department = 'Marketing';
  1. ດຶງເອົາຊື່ຂອງພະນັກງານທັງຫມົດ
  2. ດຶງເອົາຊື່ພະນັກງານຈາກພະແນກການຕະຫຼາດ
  3. ດຶງເອົາຊື່ຂອງພະນັກງານຈັດຮຽງຕາມຕົວອັກສອນ
  4. ດຶງເອົາຊື່ພະນັກງານຈາກພະແນກໄອທີ

3. ຈົ່ງເລືອກຄໍາຕອບທີ່ຖືກຕ້ອງ, ຈະໄດ້ຫຍັງຈາກ SQL query ລຸ່ມນີ້:

SELECT department, AVG(salary) AS average_salary FROM employees GROUP BY department;
  1. ເງິນເດືອນທັງໝົດຂອງແຕ່ລະພະແນກ
  2. ເງິນເດືອນສູງສຸດຂອງແຕ່ລະພະແນກ
  3. ເງິນເດືອນຕໍາ່ສຸດທີ່ແຕ່ລະພະແນກ
  4. ເງິນເດືອນສະເລ່ຍຂອງແຕ່ລະພະແນກ

4. ຈົ່ງສັງເກດເບິ່ງຕາຕະລາງ ທັງສອງຕາຕະລາງດັ່ງລຸ່ມນີ້ ແລ້ວເລືອກ SQL Query ເພື່ອ ຊອກຫາຈໍານວນການຂາຍທັງໝົດສໍາລັບພະນັກງານແຕ່ລະຄົນໃນພະແນກການຂາຍ:

Table: employees
+----+----------+------------+
| id |   name   | department |
+----+----------+------------+
|  1 |  John    |   Sales    |
|  2 |  Alice   |   Marketing|
|  3 |  Bob     |   Sales    |
+----+----------+------------+

Table: sales
+----+-----------+---------+
| id | employee_id | amount |
+----+-----------+---------+
|  1 |     1     |   1000  |
|  2 |     3     |   1500  |
|  3 |     1     |   2000  |
+----+-----------+---------+
  1. SELECT e.name, SUM(s.amount) AS total_sales
FROM employees e
JOIN sales s ON e.id = s.employee_id
GROUP BY e.name;
  2. SELECT e.name, SUM(s.amount) AS total_sales
FROM employees e
LEFT JOIN sales s ON e.id = s.employee_id
WHERE e.department = 'Sales'
GROUP BY e.name;
  3. SELECT e.name, SUM(s.amount) AS total_sales
FROM employees e
INNER JOIN sales s ON e.id = s.employee_id
GROUP BY e.name;
  4. SELECT e.name, SUM(s.amount) AS total_sales
FROM employees e
JOIN sales s ON e.id = s.employee_id
WHERE e.department = 'Sales'
GROUP BY e.id;

  1. ALTER TABLE students ADD COLUMN age INT;
  2. UPDATE TABLE students SET COLUMN age INT;
  3. ADD COLUMN age INT TO students;
  4. MODIFY TABLE students ADD COLUMN age INT;

6. ຈົ່ງເລືອກ SQL Query ທີ່ໃຊ້ເພື່ອລົບຂໍ້ມູນທີ່ສອດເສັດດ້ວຍ id ຂອງພະນັກງານ:

DELETE FROM employees WHERE id = 1;

  3. ລົບຂໍ້ມູນຂອງພະນັກງານທີ່ມີ id = 1
  4. ລົບຂໍ້ມູນຂອງພະນັກງານທີ່ມີ id ສູງສຸດສໍາລັບພະແນກ

  3. ອັບເດດລາຄາສໍາລັບພະນັກງານສູງສຸດ
  4. ອັບເດດຄໍາຕອບສູງສຸດສໍາລັບພະນັກງານທັງຫມົດ

8. ຈົ່ງເລືອກ SQL Query ທີ່ເພີ່ມບົດບັດຄຳແບບໃຫ້ແກ່ຖານຂໍ້ມູນ:

9. ຈົ່ງເລືອກ SQL Query ທີ່ຖືກຕ້ອງເພື່ອສັ່ງລາຍງານຕາມສູດທີ່ອະທິບາຍໄດ້:
