ຈົ່ງເລືອກຄໍາຕອບທີ່ຖືກຕ້ອງ SQL Query ລຸ່ມນີ້ແມ່ນໃຊ້ໄວ້ເຮັດຫຍັງ:

sql
SELECT name FROM employees WHERE department = 'Marketing';
  • A) ດຶງເອົາຊື່ຂອງພະນັກງານທັງຫມົດ
  • B) ດຶງເອົາຊື່ພະນັກງານຈາກພະແນກການຕະຫຼາດ
  • C) ດຶງເອົາຊື່ຂອງພະນັກງານຈັດຮຽງຕາມຕົວອັກສອນ
  • D) ດຶງເອົາຊື່ພະນັກງານຈາກພະແນກໄອທີ

ຈົ່ງເລືອກຜົນລັບທີ່ຖືກຕ້ອງຂອງ SQL Query ລຸ່ມນີ້ (ຊື່ຖານຂໍ້ມູນແມ່ນ student):

sql
SELECT COUNT(*) FROM students;
  • A) 0
  • B) 1
  • C) 2
  • D) 3

ຈົ່ງເລືອກຄໍາຕອບທີ່ຖືກຕ້ອງ, ຈະໄດ້ຫຍັງຈາກ Sql query ລຸ່ມນີ້:

sql
SELECT department, AVG(salary) AS average_salary FROM employees GROUP BY department;
  • A) ເງິນເດືອນທັງໝົດຂອງແຕ່ລະພະແນກ
  • B) ເງິນເດືອນສູງສຸດຂອງແຕ່ລະພະແນກ
  • C) ເງິນເດືອນຕໍາ່ສຸດທີ່ແຕ່ລະພະແນກ
  • D) ເງິນເດືອນສະເລ່ຍຂອງແຕ່ລະພະແນກ

ຈົ່ງສັງເກດເບິ່ງຕາຕະລາງ ທັງສອງຕາຕະລາງດັ່ງລຸ່ມນີ້ ແລ້ວເລືອກ SQL Query ເພື່ອ ຊອກຫາຈໍານວນການຂາຍທັງໝົດສໍາລັບພະນັກງານແຕ່ລະຄົນໃນພະແນກການຂາຍ:

sql
Table: employees
+----+----------+------------+
| id |   name   | department |
+----+----------+------------+
|  1 |  John    |   Sales    |
|  2 |  Alice   |   Marketing|
|  3 |  Bob     |   Sales    |
+----+----------+------------+

Table: sales
+----+-----------+---------+
| id |  employee_id | amount |
+----+-----------+---------+
|  1 |     1     |   1000  |
|  2 |     3     |   1500  |
|  3 |     1     |   2000  |
+----+-----------+---------+
  • A) 
    sql SELECT e.name, SUM(s.amount) AS total_sales FROM employees e JOIN sales s ON e.id = s.employee_id GROUP BY e.name;
  • B) 
    sql SELECT e.name, SUM(s.amount) AS total_sales FROM employees e LEFT JOIN sales s ON e.id = s.employee_id WHERE e.department = 'Sales' GROUP BY e.name;
  • C) 
    sql SELECT e.name, SUM(s.amount) AS total_sales FROM employees e INNER JOIN sales s ON e.id = s.employee_id GROUP BY e.name;
  • D) 
    sql SELECT e.name, SUM(s.amount) AS total_sales FROM employees e JOIN sales s ON e.id = s.employee_id WHERE e.department = 'Sales' GROUP BY e.id;

ຄໍາສັ່ງ SQL Query ໃດໃນຕົວເລືອກລຸ່ມນີ້ ທີ່ໃຊ້ເພື່ອ ເພີ່ມຖັນໃໝ່ທີ່ມີຊື່ (age) ໃສ່ຕາຕະລາງທີ່ມີຊື່ (students)

  • A) 
    ALTER TABLE students ADD COLUMN age INT;
  • B) 
    UPDATE TABLE students SET COLUMN age INT;
  • C) 
    ADD COLUMN age INT TO students;
  • D) 
    MODIFY TABLE students ADD COLUMN age INT;

ຄໍາສັ່ງໃດຕໍ່ໄປນີ້ ໃຊ້ເພື່ອແກ້ໄຂ department ຂອງ employee ທີ່ມີ ID = 101 ເປັນ “Finance” ?

  • A) 
    UPDATE employees SET department = 'Finance' WHERE id = 101;
  • B) 
    UPDATE employees SET department = 'Finance' WHERE employee_id = 101;
  • C) 
    MODIFY employees SET department = 'Finance' WHERE id = 101;
  • D) 
    ALTER employees SET department = 'Finance' WHERE id = 101;

ຈົ່ງຊອກຫາຜົນລັບທີ່ຖືກຕ້ອງຂອງ SQL Query ດັ່ງລຸ່ມນີ້:

sql
SELECT * FROM students WHERE age > 18;
  • A) ດຶງທຸກບັນທຶກຂອງນັກຮຽນທີ່ອາຍຸ 18 ປີ
  • B) ດຶງທຸກບັນທຶກຂອງນັກຮຽນທີ່ອາຍຸມາກກວ່າ 18 ປີ
  • C) ດຶງບັນທຶກທີ່ທັງໝົດຂອງນັກຮຽນທີ່ອາຍຸບໍ່ເທົ່າ 18 ປີ
  • D) ດຶງທຸກບັນທຶກຂອງນັກຮຽນທີ່ອາຍຸນ້ອຍກວ່າ 18 ປີ
